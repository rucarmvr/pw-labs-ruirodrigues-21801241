  3. Compreensão da Internet e Rede 🌐:

    • O que é um endereço de IP ?

      • Em suma um endereço de ip é um identificador que permite troca de informação entre dispositivos na rede. Um endereço de ip contém informação da localização e permite com que os dispositivos comuniquem.

      Abaixo, a nivel de experiencia é então apresentado o IP do meu laptop, do telémovel usando dados móveis e por fim o IP do servidor Heroku onde a app está alojada.



    • Percurso dos pacotes de IP : 📡

      • Um hop, ou salto, é interpretado como se fossem etapas ou dispositivos intermédios/as por onde a informação passa desde que sai da origem até ao destino.



  4. Acesso via HTTP à minha página Web 🔍:

    • HTTP GET

      • Usando o atalho Ctrl + U que simboliza um pedido HTML GET, podemos verificar que a resposta do servidor é exactamente o ficheiro que foi escrito préviamente:


    • Inspect

      • Usando o atalho Ctrl+Shift+i, que simboliza um comando de Inspect, podemos observar alguns dos pedidos eféctuados pelo servidor aos ficheiros usados na página web.

      Como é possível observar pela imagem acima podemos confirmar que a página web necessita de descarregar dois ficheiros, nomeadamente o próprio documento Html onde vem figurado todo o texto e por sua vez um ficheiro png que corresponde á imagem introduzida nesta mesma página.

      Constata-se também a ordem pela qual estes ficheiros são requisitados, o tempo que os mesmos demoraram a ser descarregados e o tamanho dos mesmos


      Explorando a ferramenta e selecionando um documento em especifico temos acesso a informações mais detalhadas desse mesmo elemento



      • Através da tab "Preview" podemos pré-visualizar o conteúdo do ficheiro em questão
